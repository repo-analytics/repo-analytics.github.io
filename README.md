# [repo-analytics.github.io](https://repo-analytics.github.io)

Thorough analytics for your GitHub repos

## Setup

Add config file

```js
// backend/secret.json
{
  "aws": {
    "accessKeyId": "",
    "secretAccessKey": ""
  },
  "github": {
    "clientID": "",
    "clientSecret": ""
  },
  "jwtSecret": ""
}
```
